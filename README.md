# kaye-protfolio

王相媛的个人作品集网站 - 艺术与科技专业

## 🎨 项目简介

这是一个为艺术与科技专业打造的沉浸式个人作品集网页，体现了跨学科、互动性和视觉创新的特点。

## ✨ 特性

- **深色渐变背景** - 从 #0A0A0A 到 #1A1A2E 的优雅过渡
- **霓虹配色方案** - 紫色 (#9D4EDD) 和电光蓝 (#00F0FF) 作为强调色
- **动态流光背景** - 随机生成的流动光线营造科技感
- **响应式设计** - 完美适配桌面端和移动端
- **交互动画** - 卡片悬停、平滑滚动、视差效果等

## 📐 页面结构

1. **Hero区域** - 全屏展示姓名、专业和宣言
2. **作品展示** - 网格布局展示6个代表作品
3. **关于/技能** - 个人陈述和技能标签展示
4. **联系区域** - 邮箱和社交媒体链接

## 🚀 使用方法

直接在浏览器中打开 `index.html` 文件即可查看。

## 🛠️ 技术栈

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript

## 📝 自定义

- 替换照片：修改 `.photo-placeholder` 中的内容为 `<img>` 标签
- 修改邮箱：在联系区域和 JavaScript 中更新邮箱地址
- 添加作品：在 `.portfolio-grid` 中添加新的 `.portfolio-card`
